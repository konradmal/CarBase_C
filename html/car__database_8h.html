Searches for cars in the database based on a specified parameter.

This function allows the user to search for cars in the database based on different criteria:

  1. Brand
  2. Model
  3. Year
  4. Engine Capacity
  5. Fuel
  6. Vehicle Type
  7. Registration Number

For each criterion, the user can choose to search for an exact match or a partial match.

Parameters
setPointer to the car database (not modified).
countNumber of cars in the database.
choiceUser's choice for the search criterion.
